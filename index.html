⚙️ Configuração do Firebase

🔧 Primeira vez? Configure o Firebase

Para sincronizar dados na nuvem, você precisa criar um projeto Firebase gratuito.

📝 Passo 1: Criar Projeto

1. Acesse https://console.firebase.google.com
2. Clique em "Adicionar projeto"
3. Nome: analisador-imobiliario
4. Desabilite Google Analytics
5. Clique "Criar projeto"

🔑 Passo 2: Pegar Credenciais

1. No painel do Firebase, clique no ícone ⚙️ (Configurações do Projeto)
2. Abra a aba "Geral"
3. Role até "Seus aplicativos"
4. Selecione "Web" e copie o objeto de configuração

🗄️ Passo 3: Ativar Realtime Database

1. No menu lateral, vá para "Realtime Database"
2. Clique "Criar banco de dados"
3. Localização: South America (São Paulo)
4. Modo de segurança: Iniciar em modo de teste
5. Clique "Ativar"

📋 Passo 4: Cole as Credenciais

Exemplo: copie tudo entre { e } da configuração do Firebase
